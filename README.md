Upgrades from VirtualLoop1
- Firmware:
  - Firmware is now remotely upgradable through the WebUI
  - Live updates in the WebUI.  No more having to refresh the pages to see detector calls.
  - More secure and rich API with faster response times
  - Camera script generator now in the WebUI and doesn't require another tool
  - Detector pulse mode which performs better for counting zones
  - Built-in call delay for advance zones
- Hardware:
  - 3x larger screen and much easier to see
  - 1Gb ethernet (vs 100Mb)
  - Faster respsonse time with support for more cameras
