Upgrades from VirtualLoop1
- Firmware is now remotely upgradable through the WebUI
- Live updates in the WebUI, no more having to refresh the pages to see detector calls or load switches
- More secure and robust API with faster response times
- Camera script generator now in the WebUI and doesn't require another tool
- Detector pulse mode which performs better for counting zones
- Call delay for advance zones
- 3x larger screen which is much easier to view
- 1Gb ethernet (vs 100Mb)
- Support for more cameras
